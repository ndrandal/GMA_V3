# Feed Simulator

Moved to its own repository: `~/Github/feed-simulator`

## Quick start

```bash
cd ~/Github/feed-simulator
node index.js --demo          # continuous feed to localhost:9001
node index.js --test --verbose # 7 deterministic scenarios
node test/smoke-test.js        # end-to-end smoke test (requires running GMA server)
```

See the feed-simulator repo for full usage and options.
